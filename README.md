# MATH1604_group_project

# Project overview
The project investigates whether the correct answers in a 100 question python quiz follows a deliberate pattern.

# Repoistory structure:
- `/scripts`: Python modules for each team member's task.
- `/data`: Storage for raw and collated quiz response text files.
- `/output`: Results, including statistical summaries and plots.
- `/reviews`: Individual Jupyter Notebooks for reflection.

# Team contributions
- **Member 1:** Data Extraction (`data_extraction_M1.py`)
- **Member 2:** Data Preparation (`data_preparation_M2.py`)
- **Member 3:** Data Analysis & Visualization (`data_analysis_M3.py`)
- **Member 4:** Lead Integrator & Repo Manager (`run_full_analysis_M4.py`)
